# sharly-chess-uac

A Sharly Chess utility to get administrator rights on Windows and perform antivirus operations.

## Usage:

### Add an exclusion to _Windows Defender_
```
sharly_chess_uac.exe --windows-defender-exclude=<dir>
```

### Add an exclusion to _Avast_
```
sharly_chess_uac.exe --avast-exclude=<dir>
```
